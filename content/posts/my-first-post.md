I started to blog in English.

I adopt [HUGO](https://gohugo.io/) to create a blog.
HUGO is a static site generators. I impressed the speed of compile.

Basic usage of HUGO is as follows.

### Create a post

```
hugo new posts/xxx.md
```

### Preview

```
hugo server -D
```

### Build

```
hugo -D
```
